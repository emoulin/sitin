# Sit{IN}

Este é um projeto desenvolvido para a IN Júnior, empresa júnior da Universidade Federal Fluminense.

## Tecnologias Utilizadas
* Java Script
* HTML
* CSS
* SASS

## Conteúdos abordados
* Responsividade
* Display flex e Grid
* Tabelas
* Formulários
* Cards
* Carrosel
